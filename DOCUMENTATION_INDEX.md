## üîç Quick Reference

### Commands

```powershell
# Verify dependencies
cd backend
python verify_dependencies.py

# Run locally
cd backend; python app.py
cd frontend; npm run dev

# Deploy to Azure
.\deploy-to-azure.ps1

# Health check
curl http://localhost:8000/health

# View logs
cat backend/logs/app.log
```

### File Locations

```
Configuration:
  backend/.env              # Environment variables
  frontend/.env             # Frontend config

Scripts:
  backend/verify_dependencies.py    # Dependency checker
  deploy-to-azure.ps1              # Azure deployment
  cleanup-azure.ps1                # Cleanup script

Workflows:
  .github/workflows/ci.yml                  # Testing
  .github/workflows/backend-deploy.yml      # Backend deploy
  .github/workflows/frontend-deploy.yml     # Frontend deploy

Utilities:
  backend/utils/azure_checkpointer.py    # Session storage
  backend/utils/azure_blob_sync.py       # Blob backup
```

### Environment Variables

```env
# Required
AZURE_OPENAI_API_KEY=...
AZURE_OPENAI_ENDPOINT=...
AZURE_DEPLOYMENT_NAME=...

# Optional Azure Services
AZURE_STORAGE_CONNECTION_STRING=...
APPLICATIONINSIGHTS_CONNECTION_STRING=...

# Feature Flags
USE_AZURE_TABLE_STORAGE=true
USE_AZURE_BLOB_STORAGE=true
USE_APPLICATION_INSIGHTS=true
```

## ‚ùì Common Questions

### "Where do I start?"
**Answer**: [QUICK_START.md](QUICK_START.md) for local dev, [AZURE_DEPLOYMENT.md](AZURE_DEPLOYMENT.md) for Azure

### "Which dependencies do I need?"
**Answer**: [DEPENDENCIES_GUIDE.md](DEPENDENCIES_GUIDE.md) - Core dependencies are required, Azure SDK is optional

### "Is the code compatible with library versions?"
**Answer**: Yes! See [COMPATIBILITY_SUMMARY.md](COMPATIBILITY_SUMMARY.md) for all fixes

### "How much does Azure cost?"
**Answer**: $0/month with free tier. See [AZURE_DEPLOYMENT.md](AZURE_DEPLOYMENT.md) cost breakdown

### "Can I run without Azure services?"
**Answer**: Yes! Set `USE_AZURE_*=false` in `.env`. See [COMPATIBILITY_SUMMARY.md](COMPATIBILITY_SUMMARY.md)

### "How do I troubleshoot issues?"
**Answer**: Check [QUICK_START.md](QUICK_START.md#-troubleshooting) and run `python verify_dependencies.py`

### "What's the agent workflow?"
**Answer**: See [README.md](README.md#-architecture) and [backend/README.md](backend/README.md)

### "How do I customize the agent?"
**Answer**: Modify files in `backend/nodes/` - see [backend/README.md](backend/README.md)
